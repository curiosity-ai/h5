# H5 Documentation

Welcome to the H5 documentation. H5 is a powerful toolset for compiling C# code into JavaScript, allowing you to build web applications using the C# language and ecosystem.

## Table of Contents

1. [Introduction](intro.md) - What is H5 and why use it?
2. [Project Structure](project-structure.md) - Understanding how H5 projects are organized.
3. [Attributes Reference](attributes.md) - Detailed guide on attributes that control code generation.
4. [Compiler Configuration](compiler-config.md) - How to configure the H5 compiler using `h5.json`.
5. [JavaScript Interop](interop.md) - How to call JavaScript from C# and vice versa.

## Getting Started

To get started with H5, you can create a new project using the `h5.template`:

```bash
dotnet new --install h5.Template
dotnet new h5 -n MyH5Project
```

For more information, see the [Introduction](intro.md).
